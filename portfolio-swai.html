Sistema para la Administración de un Centro Educativo

Este sistema permite centralizar todos los datos relativos a la gestión del centro educativo, de forma que sea más fácil su recopilación, análisis y utilización dentro del centro educativo IEBCE para la gestión de:

  • Estudiantes
  • Notas
  • Asistencias
  • Caja
  • Reportes
